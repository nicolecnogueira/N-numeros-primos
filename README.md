
## O programa

O programa imprime os primeiros "N" números primos. O valor de "N" deverá ser recebido como entrada.


## Exemplos de entradas e saídas


### Exemplo 01

Entrada: 
```console
1
```   
Saída:  
```console
2
```

### Exemplo 02

Entrada:
```console
5
```   
Saída:
```console
2
3
5
7
11
```
